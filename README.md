# 🛡️ IPhilSafe: Internet-Powered PhilSys Authentication Safe

**IPhilSafe** is an IoT-enabled smart locker system that establishes a strict, hardware-verified chain of custody for both low and high-value assets. By utilizing the **PhilSys National ID** for cryptographic identity authentication and an integrated load cell for physical mass verification, this system eliminates the "empty box" vulnerability present in traditional lockers.

---

## Features

* Simple session initiation and multi-user claiming by using native PhilSys QR code extraction and MOSIP SDK validation.
* Continuous edge-monitoring via integrated half-bridge load cells. Automatically detects unauthorized entries and compartment tampers.
* Multi-color LED visual indicators communicate state transitions directly to the user.
* Responsive cloud interface for live tracking, parameter tuning, and audit logging.
